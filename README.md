## 📖 Sobre o Projeto

O ReClick é um sistema web desenvolvido para facilitar a divulgação de eventos e a reserva de ingressos online. A plataforma permite que vendedores cadastrem eventos e que usuários possam visualizar e reservar ingressos de forma simples e organizada.

Este projeto foi desenvolvido como parte de um Projeto de Extensão do curso de Ciência da Computação, com o objetivo de aplicar na prática os conhecimentos de desenvolvimento web e sistemas de informação.

---

## 🎯 Objetivo

O objetivo do sistema é criar uma plataforma onde:
- Vendedores possam cadastrar e gerenciar eventos;
- Usuários possam visualizar eventos disponíveis;
- Usuários possam reservar ingressos;
- O sistema organize reservas e confirmações;
- Administradores possam gerenciar os eventos cadastrados.

---

## 👥 Tipos de Usuários

O sistema possui três tipos de usuários:

- **Usuário**: pode visualizar eventos e reservar ingressos;
- **Vendedor**: pode cadastrar, editar e gerenciar eventos;
- **Administrador**: pode aprovar ou rejeitar eventos cadastrados.

---

## ⚙️ Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Cadastro de eventos
- Listagem de eventos
- Reserva de ingressos
- Aprovação de reservas
- Painel do vendedor
- Painel do administrador

## 🏗 Arquitetura do Sistema

O sistema foi desenvolvido utilizando o padrão MVC (Model-View-Controller):

- **Model**: entidades do sistema (Usuário, Evento, Reserva)
- **View**: páginas HTML com Thymeleaf
- **Controller**: controle das requisições e regras do sistema
- **Repository**: comunicação com o banco de dados
- **Service**: regras de negócio

---

## 💾 Banco de Dados

O sistema utiliza o banco de dados H2 para armazenar:

- Usuários
- Vendedores
- Administradores
- Eventos
- Reservas

## ▶️ Como Executar o Projeto

1. Clonar o repositório:

git clone https://github.com/phguerrra/ReClick.git


2. Entrar na pasta do projeto:

cd re-click


3. Executar o projeto:

./mvnw spring-boot:run


4. Acessar no navegador:

http://localhost:8080
